# Vim

## Startup settings

**.vimrc:**
```bash
set encoding=utf-8
colorscheme carbonized-dark
set nowrap
set guifont=Lucida\ Console:h11
set relativenumber
set tabstop=4 softtabstop=0 noexpandtab shiftwidth=4 smarttab
```
